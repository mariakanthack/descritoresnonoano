D18: Cálculos com Números Inteiros

Adição:

Sinais iguais: soma os módulos e mantém o sinal.

Sinais diferentes: subtrai os módulos e mantém o sinal do número com maior módulo.

Exemplos:

(+5) + (+3) = +8
(+7) + (-10) = -3

Subtração:

Transformar a subtração em adição do oposto.

(+6) – (+4) = +2
(-3) – (+5) = -8
(-7) – (-2) = -5

Multiplicação:

Regras de sinal:

(+) × (+) = +
(–) × (–) = +
(+) × (–) = –
(–) × (+) = –

Exemplos:
(+3) × (-4) = -12
(-5) × (-6) = +30

Divisão:

Mesma lógica da multiplicação para os sinais.

Exemplos:
(+20) ÷ (-4) = -5
(-15) ÷ (-3) = +5

Dicas:

  • Resolva primeiro os parênteses.
  • Respeite a ordem das operações (Multiplicação, Divisão, Adição, Subtração).

D20: Resolução de Problemas com Números Inteiros

Tipos de Problemas:

1. Temperaturas:

Exemplo: Temperatura era -2°C e caiu mais 5 graus → Resultado: -7°C.

2. Altitudes:

Exemplo: Mergulhador a -15m sobe 10m → Resultado: -5m.

3. Dívidas:

Exemplo: Dívida de -80 reais, pagou 50 → Ainda falta 30 reais.

4. Lucro e Prejuízo:

Exemplo: Prejuízo de -1200, lucro de 800 → Saldo: -400.

Estratégias:

  • Ler com atenção.
  • Identificar os dados e os sinais.
  • Realizar os cálculos com cuidado.
  • Conferir a resposta no contexto do problema.

Dicas finais:

  • Sinal negativo: perda, dívida, abaixo de zero.
  • Sinal positivo: ganho, crédito, acima de zero.

D33: Analisando um Descritor em Equações do 1º Grau

O que é um Descritor?

É uma habilidade avaliada na prova.
Exemplo: Resolver e interpretar problemas com equações do 1º grau com uma incógnita.

Estrutura da Equação do 1º Grau:

Forma: ax + b = c

Onde:

  • a, b, c são números conhecidos.
  • x é a incógnita.

Passos para Resolver:

  1. Ler o problema com atenção.
  2. Identificar o que o problema está pedindo.
  3. Montar a equação.
  4. Resolver a equação.
  5. Verificar se a resposta faz sentido.

Exemplo de Problema:

João tem o triplo da idade de Ana. Daqui a 4 anos, a soma das idades será 40 anos. Qual a idade atual de Ana?

Resolução:

x = idade de Ana.
João = 3x.
Equação: (x + 4) + (3x + 4) = 40
Resultado: x = 8.

Dicas para Prova:

  • Leia com atenção.
  • Identifique a incógnita.
  • Monte a equação.
  • Faça os cálculos com calma.
  • Revise antes de marcar a resposta.
